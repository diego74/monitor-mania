## Fase 3: Test nuevo compuesto con ramificación (REQUISITOS ACTUALIZADOS)

### Objetivo
Crear un test que combine manía + mood + depresión, con lógica de branching avanzada basada en:
- Variables de contexto (edad, ciclo menstrual)
- Mezcla aleatoria diaria
- Preguntas distractoras y enmascaramiento
- Ramificación condicional por scores en tiempo real
- Detección automática de inconsistencias
- Output normalizado para mapa de calor 3-ejes

### Especificaciones técnicas del prompt

#### 2. Arquitectura de Preguntas (Anti-Mecánica)

**A. Mezcla Aleatoria Diaria**
- El orden de preguntas debe rotar cada día (basado en fecha).
- Implementación: `shuffleQuestions(questions, dateString)` con seed determinista.

**B. Preguntas Distractoras**
- Insertar 1 pregunta neutra (clima, música) cada ~6 preguntas para resetear patrón cognitivo.
- Implementación: En `compositeTestConfig.js`, definir lista de `distractorQuestions`.

**C. Enmascaramiento**
- No preguntar por el síntoma directo.
- Ejemplo: En lugar de "¿Estás grandioso?", usar "¿Sientes que el ritmo del mundo es demasiado lento para ti?".
- Implementación: Las preguntas en el JSON ya están enmascaradas. Revisar las existentes y agregar variants si es necesario.

#### 3. Lógica de Ramificación (Jump Logic)

**Nivel 1 - Hitos Biológicos (siempre)**
- Pregunta obligatoria: horas de sueño (input: hora acostado / hora despertado).
- Implementación: Agregar al inicio del test:
  ```javascript
  { id: 'sleep_input', type: 'time_range', labels: ['Hora dormir', 'Hora despertar'] }
  ```
- Calcular sueño en horas → afecta branching.

**Nivel 2 - Dimensiones**
- Escalas 1-5 camufladas en frases de esfuerzo.
- Las preguntas actuales ya usan escala 0-4 (5 opciones).
- Calcular scores parciales en tiempo real.

**Nivel 3 - Profundización Condicional**
- Disparar preguntas extra si:
  - `sleepHours < 5` → validación cruzada (ej: "¿Alguien te ha comentado que hablas muy rápido?")
  - `cualquier dimensión ≥ 4` → validación específica
- Implementación: En `CompositeTest.js`, tras cada respuesta, calcular scores de dimensiones disponibles y agregar preguntas de `validationQuestions` según condiciones.

#### 4. Detección de Inconsistencias (Algoritmo de Veracidad)

Reglas:
- **Regla A**: Si `sleepHours < 4` Y `energía_reportada ≥ 3` → alerta "Manía probable (insomnio con alta energía)".
- **Regla B**: Si `conflict = 0` Y `irritabilidad = 4-5` → alerta "Irritabilidad contenida (sin expresion externa)".
- **Regla C**: Si `mood_score depresión ≥ 3` pero `patient` reporta "bien" → alerta "Disforia congruente vs incongruente".
- Implementación: Función `detectInconsistencies(answers, scores)` que retorna array de `{type, message, severity}`.

#### 5. Output de Datos

**Normalización**:
- Todas las respuestas a escala 0-4 (ya está).
- Scores por dimensión: promedio 0-4.
- Overall score: combinación ponderada.

**Mapa de Calor 3 Ejes** (para gráficos):
- **Mente**: `distractibility` + `flight_of_ideas` + `pressured_speech`
- **Cuerpo**: `energy` + `sleep` + `psychomotor` (si existe)
- **Social**: `conflict` + `irritability` + `grandiosity`
- Implementación: Función `computeTripleAxis(byDimension)` que devuelve `{ mind: 0-4, body: 0-4, social: 0-4 }`.

```javascript
export const distractorQuestions = [
  {
    id: 'dist_weather',
    question: '¿Cómo describirías el clima de hoy?',
    options: ['Soleado', 'Nublado', 'Lluvioso', 'Ventoso', 'No lo sé'],
    measures: 'distractor',
    isDistractor: true
  }
];

export function getCompositeQuestions(role, dateString) {
  // 1. Seleccionar base según rol
  const base = role === 'caregiver' ? caregiverQuestions : patientQuestions;
  const mood = moodQuestions;
  const depression = role === 'caregiver' ? depressionCaregiverQuestions : [];

  // 2. Aplanar y etiquetar origen
  let all = [
    ...base.map(q => ({ ...q, _origin: 'mania' })),
    ...mood.map(q => ({ ...q, _origin: 'mood' })),
    ...depression.map(q => ({ ...q, _origin: 'depression' }))
  ];

  // 3. Insertar distractores (cada 6 preguntas aproximadamente)
  const inserted = [];
  let distractorIndex = 0;
  for (let i = 0; i < all.length; i++) {
    inserted.push(all[i]);
    if ((i + 1) % 6 === 0 && distractorIndex < distractorQuestions.length) {
      inserted.push({ ...distractorQuestions[distractorIndex], _origin: 'distractor' });
      distractorIndex++;
    }
  }

  // 4. Mezclar con seed diaria (fecha como seed)
  const seed = dateString.split('-').join('').hashCode(); //需要 hash function
  const shuffled = FisherYatesShuffle(inserted, seed);

  return shuffled;
}
```

```javascript
import { useState, useEffect } from 'react';
import { getCompositeQuestions, branchingRules } from '../data/compositeTestConfig';
import { calcCompositeSeverity } from '../utils/compositeScoring';
import { saveComposite } from '../services/storage';

export default function CompositeTest() {
  const { role } = useApp();
  const today = new Date().toISOString().split('T')[0];
  const [questions, setQuestions] = useState([]);
  const [answers, setAnswers] = useState({});
  const [activeBranching, setActiveBranching] = useState([]); // IDs de preguntas ramificadas activas
  const [context, setContext] = useState({ sleepHours: null, maniaScore: 0, moodScore: 0, depressionScore: 0 });
  const [inconsistencies, setInconsistencies] = useState([]);

  // Inicializar preguntas base
  useEffect(() => {
    const base = getCompositeQuestions(role, today);
    setQuestions(base);
  }, [role, today]);

  // Calcular contexto cada vez que cambian respuestas relevantes
  useEffect(() => {
    const ctx = computeContext(answers);
    setContext(ctx);
    
    // Evaluar reglas de branching
    const triggered = [];
    branchingRules.forEach(rule => {
      if (rule.condition(ctx)) {
        triggered.push(...rule.questions.map(q => q.id));
      }
    });
    setActiveBranching(triggered);

    // Detectar inconsistencias
    const inc = detectInconsistencies(answers, ctx);
    setInconsistencies(inc);
  }, [answers]);

  function handleAnswer(questionId, value) {
    setAnswers(prev => ({ ...prev, [questionId]: value }));
  }

  async function handleSubmit() {
    // Validar que todas las preguntas base estén respondidas
    const baseIds = questions.filter(q => !q.isDistractor && !q.isBranching).map(q => q.id);
    const allBaseAnswered = baseIds.every(id => answers[id] !== undefined);
    if (!allBaseAnswered) {
      alert('Respondé todas las preguntas base');
      return;
    }

    const scores = calcCompositeSeverity(answers, questions);
    const payload = {
      timestamp: new Date().toISOString(),
      date: new Date().toLocaleDateString('es-PE'),
      patientId: 'daniela',
      testType: 'composite',
      submittedByRole: role,
      severity: scores.overall,
      scoresBySection: scores.byModule,
      raw: answers,
      branchingLog: activeBranching.map(id => ({ questionId: id, triggered: true })),
      inconsistencies: inconsistencies,
      context: context
    };
    await saveComposite(payload);
  }

  // Renderizar: preguntas base + ramificadas activas
  const displayQuestions = [...questions, ...branchingQuestions.filter(q => activeBranching.includes(q.id))];

  return (
    <Card>
      {/* Mostrar alertas de inconsistencia */}
      {inconsistencies.length > 0 && (
        <Alert variant="warning">
          {inconsistencies.map((inc, i) => (
            <p key={i}>⚠️ {inc.message}</p>
          ))}
        </Alert>
      )}

      <TestForm
        questions={displayQuestions}
        answers={answers}
        onAnswer={handleAnswer}
        onSubmit={handleSubmit}
      />
    </Card>
  );
}
```

#### `src/utils/compositeScoring.js`

```javascript
import { calcSeverity } from '../data/questions';
import { calcMoodSeverity } from './scoring';

export function calcCompositeSeverity(answers, allQuestions) {
  // Separar preguntas por módulo
  const maniaQ = allQuestions.filter(q => q._origin === 'mania');
  const moodQ = allQuestions.filter(q => q._origin === 'mood');
  const depressionQ = allQuestions.filter(q => q._origin === 'depression');

  const maniaScore = maniaQ.length ? calcSeverity(answers, maniaQ).overall : null;
  const moodScore = moodQ.length ? calcMoodSeverity(answers, moodQ).overall : null;
  const depressionScore = depressionQ.length ? calcSeverity(answers, depressionQ).overall : null;

  // Weighted overall: mania 0.4, mood 0.35, depression 0.25 (ajustable)
  const weights = { mania: 0.4, mood: 0.35, depression: 0.25 };
  const valid = [maniaScore, moodScore, depressionScore].filter(v => v !== null);
  const weightsUsed = valid.map((v, i) => [weights.mania, weights.mood, weights.depression][i]);
  const totalWeight = weightsUsed.reduce((a, b) => a + b, 0);
  const overall = valid.reduce((sum, score, i) => sum + score * weightsUsed[i], 0) / totalWeight;

  return {
    byModule: { mania: maniaScore, mood: moodScore, depression: depressionScore },
    overall: parseFloat(overall.toFixed(2))
  };
}
```

### Hash para mezcla determinista
```javascript
function hashCode(str) {
  let hash = 0;
  for (let i = 0; i < str.length; i++) {
    hash = ((hash << 5) - hash) + str.charCodeAt(i);
    hash |= 0;
  }
  return Math.abs(hash);
}

function shuffleArray(array, seed) {
  const rand = (s) => {
    const x = Math.sin(s++) * 10000;
    return x - Math.floor(x);
  };
  const arr = [...array];
  for (let i = arr.length - 1; i > 0; i--) {
    const j = Math.floor(rand(seed) * (i + 1));
    [arr[i], arr[j]] = [arr[j], arr[i]];
    seed++;
  }
  return arr;
}
```
